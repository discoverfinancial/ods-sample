# End of Life Guide

ODS App's Endoflife data comes from the website [endoflife.date](https://endoflife.date/). This data provides information on software end of life dates and support lifecycles. ODS App's End of life table has the following schema:

| Variable      | Variable Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
|---------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Name          | Software Name                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| Category      | Software Category                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| Releases      | <table><tbody><tr><td>Release</td><td>Release (version) number</td></tr></tbody><tbody><tr><td>Release Date</td><td>Date that Software Version was released</td></tr></tbody><tbody><tr><td>Maintained</td><td>Boolean of whether software version still has some level of support.<br/>This can be any level of support, even extended support.</td></tr></tbody><tbody><tr><td>EOL Date</td><td>End of life date for the release cycle</td></tr></tbody><tbody><tr><td>EOAS Date</td><td>End of active support date for the release cycle</td></tr></tbody><tbody><tr><td>EOES Date</td><td>End of extended support date for the release cycle</td></tr></tbody></table> |
| Last Modified | Date that the End of life entry was last modified within ODS App                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
To view all end of life information for an entry, double click into the row.
