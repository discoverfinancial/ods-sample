ODS App's Notebooks tool allows you to create server-side JavaScript notebooks. This allows you to build and document complex queries that run completely on the server, while presenting only the information you want in the browser.

ODS App Notebooks are made up of cells. Currently there are 5 different types of cells supported, which you can change via the type dropdown. Those 5 types are JavaScript, Markdown, Search Software, React Code, and Input:
- JavaScript cells allow you to write code that can be executed on server.
- Markdown cells allow you to show information in markdown format, which can behave as comments or descriptions interspersed throughout your notebook.
- Search Software cells allows you to search through ODS App's software data.
- React Code cells allow you to create react components.
- Input cells allow you to ask a user to input a value for a variable that will be used in other code cells.



Cells can be interacted with and executed individually, by pressing the play button at the top of the cell. You can also move cells, insert new cells, and delete cells with the other options next to the play button. When you finish working on the notebook and want to share the results, feel free to click on the presentation mode checkbox in the top right corner of the page. This allows you to hide the code and only display the results throughout for a more aesthetically simple notebook. Make sure to press the Save Notebook button at the bottom of the page to save your work.



For information on additional methods available for use in Notebooks, press the Notebook Help button on the right side of the page.
